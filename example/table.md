# Tables
* You can have tables too!
* This:
```markdown
| \(x\) | \(x^2\) | \(x^3\) |
|-------|---------|---------|
| 0     | 0       | 0       |
| 1     | 1       | 1       |
| 2     | 4       | 8       |
```
will render as:

| \(x\) | \(x^2\) | \(x^3\) |
|-------|---------|---------|
| 0     | 0       | 0       |
| 1     | 1       | 1       |
| 2     | 4       | 8       |
